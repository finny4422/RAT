# UI Specification

Tabs:
1. Activities
2. Create
3. Reports

Activities Screen:
- Show only activities relevant today
- Hide completed activities
- Full card color changes based on status

Card Layout:
Left:
- Title
- Caption
- Due Time

Right:
- Checkbox

Colors:
White = Pending
Yellow = Due Soon
Red = Missed

Sort:
Missed
Due Soon
Pending

Within groups:
Nearest due time first

Widget:
Shows top 2 activities
Same sorting logic as Activities tab
Supports checkbox completion
